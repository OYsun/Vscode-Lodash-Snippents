
# 安装 

* [vscode插件地址](https://marketplace.visualstudio.com/items?itemName=oysun.Lodash)
```javascript
ext install Lodash
```
# 支持的文件格式
- javascript
- coffeescrip
- typescript
- javascriptreac
- vue
